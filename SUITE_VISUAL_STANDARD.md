# Enterprise AI & IT Leadership Suite — Visual Standard

This app belongs to the Enterprise AI & IT Leadership Suite.

## Visual Identity

- Premium enterprise SaaS
- Dark navy / slate background
- Executive dashboard aesthetic
- Sidebar + topbar structure
- Clean KPI cards
- Rounded 2xl containers
- Subtle gradients
- Cyan / blue primary accent
- Emerald for value and success
- Amber for caution
- Rose for risk or blockers
- Violet / indigo for intelligence and AI insight layers
- No emojis inside product UI
- No generic startup template look
- No childish or consumer-style UI

## Shared UX Pattern

Every core app should include:

- Sidebar navigation
- Topbar with EN/ES selector
- Executive page header
- KPI summary cards
- Main working view
- Detail modal or inspector
- Decision intelligence / explanation panel
- Settings / Demo Boundaries
- Clear simulated AI / no API key / no confidential data disclosure

## Typography

- Executive, compact and readable
- Strong uppercase labels for metadata
- Clear hierarchy:
  - page title
  - section title
  - metric label
  - table/body text
  - helper/disclaimer text

## App Identity

Each app may use a specific accent and visualizations, but must still feel part of the same suite.

Do not copy screens blindly. Reuse the visual language, not the exact product structure.
